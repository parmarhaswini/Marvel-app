## 🚀 Installation

### Clone the Repository
```bash
git clone https://github.com/parmarhaswini/Marvel-app.git
cd Marvel-app
```

### CocoaPods (if applicable)
```bash
pod install
```

### Using Xcode
1. Open `Marvel-app.xcodeproj` in Xcode
2. Select the target device
3. Press `Cmd + R` to build and run
